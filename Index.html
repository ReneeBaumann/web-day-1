HTML

What is HTML?

The first web page was written in HTML (HyperText Markup Langauge). It is the standard markupp language used to create web pages. It structures content on the web, allowing browsers to display text, images, and other elements

How to use HTML

HTML uses tags to define elemtns. Tags are enclosed in angle brackets. Elemnts can include headins, paragraphs, link, images, and lists.
HTML documents are structured with basic layout, inlcuding a HEAD section for metadata and a BODY section for content.
